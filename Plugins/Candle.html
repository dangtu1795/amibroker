Candle.DLL - Candlestick functions plug-in

Copyright (C)2001 by Tomasz Janeczko/Amibroker.com

The candlestick functions plug-in is rather a very simple sample plug-in than a real tool. Currently it exposes only 8functions.
All functions have "Cd" prefix appended to avoid name conflicts with built-in functions.

  1. CdWhite( bodyfactor = 0.4, rangefactor = 0) - returns True (1) when white candle appears, False (0) otherwise.
  2. CdBigWhite( bodyfactor = 0.9, rangefactor = 1.5) - returns True (1) when a big white candle appears, False (0) otherwise.
  3. CdBlack( bodyfactor = 0.4, rangefactor = 0) - returns True (1) when black candle appears, False (0) otherwise.
  4. CdBigBlack( bodyfactor = 0.9, rangefactor = 1.5) - returns True (1) when a big black candle appears, False (0) otherwise.
  5. CdDoji( threshold = 0.05 ) - returns True (1) when doji is detected, False (0) otherwise. The threshold argument defines maximum allowed difference between open and close as a fraction of high-low range.
  6. CdHammer( rangefactor= 1.1 ) - returns True (1) when hammer is detected, False (0) otherwise. This function checks if Close is higher than half the distance between high and low and the candle is black. In addition to that the body size of the candle must be greater rangefactor * average |close-open| range from 15 bars, and open has to be located not lower than 10% of candle height from the high price.
  7. CdBearishEngulfing( bodyfactor = 0.4, rangefactor = 0.5) returns True (1) when Bearish engulfing pattern is detected, False (0) otherwise.
  8. CdBullishEngulfing( bodyfactor = 0.4, rangefactor = 0.5) returns True (1) when Bullish engulfing pattern is detected, False (0) otherwise.

Notes:

The bodyfactor argument controls the minimum accepted size of candle's body as a fraction of high-low range.
The rangefactor argument controls the minimum accepted size of candle's body compared to 15-bar simple moving average of |close-open| range.
